
#### COCO

**[COCO](https://cocodataset.org/#home)** is a large-scale object detection, segmentation, and captioning dataset. 

数据集现在有3种标注类型：object instances（目标实例）, object keypoints（目标上的关键点）, 和 image captions（看图说话），使用 JSON 文件存储
 为了更方便的使用模型

可以使用 [cocoapi](https://github.com/cocodataset/cocoapi) 查看数据集
